# Video Game Sales Analysis Project

## Project Overview
This project analyzes a video game sales dataset to uncover trends in game sales across different platforms, genres, and regions. The project includes data cleaning, exploratory data analysis, SQL analysis, and interactive Power BI dashboards.

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Excel

## Project Workflow
1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL queries for business insights
4. Power BI dashboard creation

## Key Insights

- Action and Sports genres generate the highest global sales.
- PlayStation and Nintendo platforms dominate the market.
- North America contributes the largest share of global sales.
